FEATURES



CLOUD

Our project has been containerize using Docker. Moreover this has been hosted in an x86 enviornment using Virtual Linux containers on Baremetal emulation device.


NLP

We use NLP to use sponsor content data and train a Gensim Doc2Vec model for anomaly detection. Outliers are content and the rest is sponsored text.


STREAMING

The user will add desired YouTube url as input and the processed video will be streamed in the SponsEnd website itself.This video will be free of the sponsored segments.


SPONSORS

Sponsorships take up time when placed in videos delivering crucial information. We don't mean to infringe on their agency, we want users to have a good viewing experience.


HMI

The team has done extensive research and user interviews to understand issues of using YouTube.SponsEnd uses HTML, CSS and Vanilla JS.
